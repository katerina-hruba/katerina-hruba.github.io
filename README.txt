## 🚀 Jak nahrát na GitHub Pages:

### Krok 1: Vytvoření GitHub účtu
1. Jděte na https://github.com
2. Klikněte "Sign up" a vytvořte účet
3. Ověřte email

### Krok 2: Vytvoření repozitáře
1. Po přihlášení klikněte na "+" vpravo nahoře
2. Vyberte "New repository"
3. Název: **vasejmeno.github.io** (nahraďte "vasejmeno" vaším GitHub username)
4. Zaškrtněte "Public"
5. Klikněte "Create repository"

### Krok 3: Nahrání souborů
1. V repozitáři klikněte "Add file" > "Upload files"
2. Přetáhněte VŠECHNY soubory z tohoto ZIP:
   - index.html
   - style.css
   - app.js
   - K_Hruba_PST_PBSP.jpg
3. Napište commit message: "Přidání webových stránek"
4. Klikněte "Commit changes"

### Krok 4: Aktivace GitHub Pages
1. Klikněte na "Settings" (nastavení)
2. V levém menu najděte "Pages"
3. V sekci "Source" vyberte:
   - Branch: **main**
   - Folder: **/ (root)**
4. Klikněte "Save"

### Krok 5: Ověření
- Počkejte 2-3 minuty
- Váš web bude dostupný na: **https://vasejmeno.github.io**
- Otevřete v prohlížeči a zkontrolujte

## 🌐 Vlastní doména (volitelné):

Pokud chcete vlastní doménu (např. katerina-hruba.cz):

1. Kupte doménu u poskytovatele (Wedos, Forpsi)
2. V GitHub Pages nastavení zadejte "Custom domain"
3. U registrátora domény nastavte DNS záznamy podle GitHub návodu
4. Zapněte "Enforce HTTPS"

## ✏️ Jak upravit obsah:

### Změna textu:
- Upravte soubor **index.html**
- Najděte příslušný text a změňte ho
- Uložte a nahrajte znovu na GitHub

### Změna barev:
- Upravte soubor **style.css**
- Na začátku najdete sekci `:root` s barvami
- Změňte hodnoty jako `--color-primary: #5D7285;`

### Změna fotografie:
- Nahrajte novou fotografii se STEJNÝM názvem: **K_Hruba_PST_PBSP.jpg**
- Nebo upravte v index.html řádek: `<img src="nazev-fotky.jpg">`

## 🆘 Podpora:

Pokud narazíte na problém:
1. Zkontrolujte, že jsou všechny soubory nahrané
2. Ověřte, že fotografie má správný název
3. Zkuste vyprázdnit cache prohlížeče (Ctrl+F5)
